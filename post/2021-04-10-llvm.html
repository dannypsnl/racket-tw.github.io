[Scheme] 安裝 Racket-llvm

Yoxem <yoxem.tem98@nctu.edu.tw>

  • Clone git repo：

    git clone git@github.com:bubba/racket-llvm.git

  • 包做 .zip：
    raco pkg create --binary racket-llvm

  • 安裝：
    raco pkg install racket-llvm.zip
